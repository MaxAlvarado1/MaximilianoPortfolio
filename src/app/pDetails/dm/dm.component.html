Description

University collabrated project with 3 other team members. I was team leader overseeing everyone's work, making sure to be coordinating people's roles, assemble meetings, and build goals to reach deadlines.

Project was a full-stack (frontend, backend, and database) web development focused a lot more on the functions of the databases system.

Database System

Tools used:

  • Microsoft SQL Server Management Studio
  • Microsoft Azure
  • ASP.NET

Methods and Features:

  • Stored Procedures ONLY
  • Triggers
  • Data Entry Forms
  • Entities and Relationships
  • Data Reports
  • Autorization Roles

Front End - ASP.NET

All tables, entry forms, and logins pull from different data tables using SQL queries in stored procedures.

Front end work is basic and simple, as the focus was more on the backend and creating a complex database system.
