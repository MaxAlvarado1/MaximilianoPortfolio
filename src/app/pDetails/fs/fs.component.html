Description

A full-stack (frontend, backend, and database) web development for a local automotive salvage yard showcasing the yard's information and inventory.

Inlcudes an Admin role, which can add, edit, and delete the yard's inventory database.

Front End - Angular 15

Tools used:

  • Visual Studio Code
  • Node.js

Vistor's Dashboard:

  • Access to view the store's Inventory Database
  • Access to Google Maps API to show store's location
  • Easily navigate through store's information and parts' details.
  • Check this quick demo video.

Admin's Dashboard:

  • Accessed through a secured login.
  • Access to view, add, edit, and delete the store's Inventory Database.
  • Check this quick demo video.

Back End and Database System - ASP.NET Web API and SQL Servers

Tools used:

  • Visual Studio
  • POSTMAN
  • Microsoft SQL Server Management Studio

Methods:

  • HTTP Method Responses in JSON
  • APIs: GET, POST, PUT, and DELETE
  • API to upload photos
  • Stored Procedures ONLY
